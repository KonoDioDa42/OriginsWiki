Voidsight Potion

The Voidsight Potion is a buff potion that can be crafted using a Fireblossom, Night Owl Potion and a Duskarp. When consumed, it provides the Voidsight buff, which increases the apparent light level provided by all light sources on screen and causes the player to emit a lot of light themselves. This effect will last for 6 minutes, but can be canceled anytime by right-clicking the icon.
