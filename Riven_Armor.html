Riven Armor

Riven armor is an armor set consisting of the Riven Mask, Riven Coat and Riven Pants. In order to craft the full set the player will need 60 Encrusted Bars (180 Encrusted Ore) and 45 Riven Carapace.

Riven armor pieces are interchangable with pieces from the Ancient Riven armor set.

A full set of Riven armor provides the following benefits:

- +19 defense
- +18% attack speed
Set bonus
- +10% Torn severity
