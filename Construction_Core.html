Construction Core

The Construction Core is a post-Moon Lord, Eyndum-tier accessory that is crafted from materials obtained through the Test of the Ancients event at an Elderforge. It can only be equipped in a dedicated core slot provided by a full set of Eyndum Armor that appears to the left of the player's equipment section. It provides the following boosts:

- +100% tile placement speed
- +150% wall placement speed
- +200% mining speed
- +30 tile range
- +40 block placement range
