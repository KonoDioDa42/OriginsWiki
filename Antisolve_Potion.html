Antisolve Potion

The Antisolve Potion is a buff potion that can be crafted using Bottled Brine, Brineglow and Peat Moss. When consumed, it provides the Unsolvable buff, which provides immunity to the Toxic Shock and Torn debuffs. This effect will last for 8 minutes, but can be canceled anytime by right-clicking the icon.
