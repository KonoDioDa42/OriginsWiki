Nova Armor

Nova armor is a post-Moon Lord armor set consisting of the Nova Faceshield, Nova Vest and Nova Leggings. In order to craft the full set the player will need 36 Luminite Bars (144 Luminite) and 45 Nova Fragments.

A full set of Nova armor provides the following benefits:

- +48 defense
- +12% explosive damage
- +12% explosive critical strike chance
- +50% explosive velocity
- -45% self-damage
- +15% movement speed
Set bonus
- +100% explosive blast radius
- 50% chance to duplicate explosive projectiles
- Explosive projectiles home to enemies
