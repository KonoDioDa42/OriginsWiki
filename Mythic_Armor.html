Mythic Armor

Mythic armor is an armor set consisting of the Mythic Skull, Mythic Shell and Mythic Leggings. In order to craft the full set the player will need 50 Bones, 30 Clouds and 30 Feathers.

A full set of Mythic armor provides the following benefits:

- +12 defense
- +15% explosive velocity
- +45% jump speed
- +20% movement speed
Set bonus
- Increased explosive damage by 20% of the player's current speed
- Increased explosive velocity by 50% of the player's current speed
