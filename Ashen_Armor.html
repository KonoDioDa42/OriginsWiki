Ashen Armor

Ashen armor is an armor set consisting of the Ashen Helmet, Ashen Breastplate and Ashen Greaves. In order to craft the full set the player will need 60 Sanguinite Bars (180 Sanguinite Ore) and 45 NE-8.

A full set of Ashen armor provides the following benefits:

- +19 defense
- +8% damage
- +8% attack speed
- +8% critical strike chance
Set bonus
- +15% knockback and knockback resistance
- -40% self-damage
