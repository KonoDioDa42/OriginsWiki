Amber Armor

Amber armor is a Hardmode armor set consisting of the Amber Exploder Goggles, Amber Exploder Resin and Amber Exploder Guards. In order to craft the full set the player will need 24 Amber, 9 Souls of Night and 72 Carburite.

A full set of Amber armor provides the following benefits:

- +21 defense
- -20% self-damage
- +10% movement speed
Set bonus
- -38% explosive blast radius
- All explosives are preserved in an amber resin and release damaging amber shards upon detonation
- Amber shards inflict Slow, halve enemy defense and heal the player for any life lost from self-damage
