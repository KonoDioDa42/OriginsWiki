Void Lock

The Void Lock is a consumable item that is crafted from a Chest Lock, Bones, Jungle Spores and any evil tissue. Pressing the Use button on certain types of Chests while holding it will turn the Chest into a locked Chest and consume the Void Lock. Chests locked by this can only be opened by the player that placed the Void Lock on them, providing security from griefing in multiplayer. The chest can be unlocked by using the Interact button without any item being needed.
