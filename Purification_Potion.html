Purification Potion

The Purification Potion is a buff potion that can be crafted using Ectoplasm, Moonglow and a Mojo Flask. When consumed, it provides the Willed buff, which provides immunity to and removes all current progress of any kind of Assimilation. This effect will last for 6 minutes, but can be canceled anytime by right-clicking the icon.
