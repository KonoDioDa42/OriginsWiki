Scavenger Armor

Scavenger armor is an armor set consisting of the Scavenger Goggles, Scavenger Body Armor and Scavenger Greaves. In order to craft the full set the player will need 42 Sanguinite Ore and 66 Biocomponent 10.

A full set of Scavenger armor provides the following benefits:

- +10 defense
- +8% explosive critical strike chance
- -25% self-damage
- +15% explosive velocity
Set bonus
- All attacks inflict Impeding Shrapnel
- 35% chance for additional Scrap from any source
