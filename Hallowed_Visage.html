Hallowed Visage

The Hallowed Visage is a helmet armor piece to serve as the Demolitionist-equivalent for Hallowed armor. Equipping this as part of a full set does not change the set bonus provided and only offers basic combat stats.

The Hallowed Visage provides the following benefits:

- +3 defense
- +15% explosive velocity
- +8% explosive critical strike chance
