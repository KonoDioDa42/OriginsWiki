Encrusted Armor

Encrusted armor is an armor set consisting of the Encrusted Mask, Encrusted Coat and Encrusted Pants. In order to craft the full set the player will need 42 Encrusted Ore and 66 Bud Barnacle.

A full set of Encrusted armor provides the following benefits:

- +10 defense
- +10% minion damage
- +1 minion slot
- +15% jump speed
Set bonus
- +30% minion damage when above half health
