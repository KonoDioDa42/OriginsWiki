Peat Moss

"The Demolitionist might find this interesting..."

Peat Moss is a pre-Hardmode material and block type found in the Brine Pool. It envelops the outermost parts of each cell within the Brine Pool, but it can be grown on any mud block given it is submerged. It can additionally be dropped by many enemies in the Underground Jungle. Peat Moss can be sold to the Demolitionist NPC to unlock some explosive items that which he can sell.
