Magic Brine Dropper

The Magic Brine Dropper is a Hardmode item used to place "brine emissions" on ceilings. The smoky effect is purely visual, and has no functional use. The Magic Brine Dropper is crafted with an Empty Dropper at Brine Water, with the Crystal Ball nearby. It smoke effect can be removed by destroying the block its was place under, but will not drop any item.
