Missing File

The Missing File is a Hardmode magic weapon found in Defiled Dungeon Chests. Selecting it in the hotbar while enemies are nearby causes several icons to pop up for the player to click on with the cursor. Whenever the correct correspondent icon is selected for the enemy the player desires to attack, all copies of that enemy on screen get dealt substantial damage from spikes growing out of their chest, and a meowing noise can be heard. When the wrong icon is selected, it does nothing and changes the icons that are on screen.

The best modifier is Mythical.
