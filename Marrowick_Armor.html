Marrowick Armor

Marrowick armor is an armor set consisting of the Marrowick Cove, Marrowick Coat and Marrowick Greaves. In order to craft the full set the player will need 75 Marrowick.

A full set of Marrowick armor provides the following benefits:

- +4 defense
Set bonus
- +1 defense
