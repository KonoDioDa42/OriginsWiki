Ancient Felnum Armor

Ancient Felnum armor is an armor set that reproduces the old design, and is otherwise identical in all ways to the current Felnum armor set. Each piece is interchangeable with the equivalent Felnum armor piece, while still granting the same set bonus.

Each piece has a 1/66 (1.52%) chance of being dropped by most Felnum enemies.
